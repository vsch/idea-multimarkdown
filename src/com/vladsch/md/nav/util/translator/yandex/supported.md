## Error Codes

| Value |                        Description                        |
|-------|-----------------------------------------------------------|
| 200   | Operation completed successfully                          |
| 401   | Invalid API key                                           |
| 402   | Blocked API key                                           |
| 403   | Invalid API key                                           |
